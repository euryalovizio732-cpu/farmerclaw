# 抖音三农商家 · 全流程内容自动化 Agent
## 详细开发计划 v2.0

> **一句话定位**：帮抖音三农商家用 AI Agent 替代「内容运营」这个岗位，把每天要花 3-5 小时做的视频选题+口播稿+直播话术，压缩到 3 分钟自动完成。

---

## 一、核心痛点锁定（MVP 只打一个）

### 选定痛点：**抖音三农商家每天不知道拍什么、不会写口播稿**

#### 痛点拆解

```
三农商家在抖音的日常：
  ├─ 每天要发 1-3 条视频（算法要求）
  ├─ 每次直播 2-4 小时（不能冷场）
  └─ 但他们是：农民/合作社负责人，不是运营

他们现在怎么解决：
  ├─ 自己硬憋（质量差、放弃率高）
  ├─ 雇运营（3000-8000元/月，三农商家大多付不起）
  └─ 套模板（内容同质化，流量越来越差）
```

#### 为什么是蓝海（Agent 替代人工）

| 对比维度 | 现有解法 | AI Agent 方案 |
|---------|---------|--------------|
| 通用 AI 工具（剪映/文心） | 不懂三农场景，生成内容不接地气 | 三农专属知识库，产地背书/坏果包赔/节气营销全内置 |
| 人工运营 | 3000-8000元/月，小商家负担不起 | 498元/月，成本降 90% |
| 自己摸索 | 日均 3-5 小时，低效 | 3 分钟出全套，解放双手 |
| 竞品 AI SaaS | 聚焦大盘电商，无三农垂直产品 | **当前市场空白，零直接竞争** |

#### 核心替代逻辑

```
被替代的岗位：内容策划 + 文案撰写（每月 5000-8000 元）
替代方式：商家输入「今天卖什么」→ Agent 自动输出「今天发什么、怎么直播」
替代程度：95%（仅保留商家录制/出镜环节）
```

## 二、产品定义（MVP 最小边界）

### 核心服务：「今日内容包」一键生成

**商家每天打开产品，填一个表（2 分钟）→ 获得：**

```
📅 今日内容包（抖音专属）
  ├─ 3 条短视频选题（含拍摄角度建议）
  ├─ 3 条完整口播稿（30 秒，含开场钩子）
  ├─ 今日直播完整话术（2 小时模块化，可循环用）
  └─ 5 个今日发布话题标签（对齐平台热点）
```

### 不做的事（MVP 红线）

- ❌ 不做自动发布（需要抖音企业号 API，门槛高）
- ❌ 不做数据分析（MVP 后做）
- ❌ 不做评论自动回复（Phase 2）
- ❌ 不做视频剪辑（不碰音视频处理）
- ❌ 不适配拼多多/惠农网（先打透抖音）

### Phase 2 · 抖音专项深化（当前阶段，Day11-Day20）

**核心目标：把「今日内容包」做到让商家每天离不开**

#### Day11-Day13：抖音内容 Agent 专项优化

**任务 A：抖音选题 Agent（新增）**

```
输入：品类 + 产品 + 当前节气/节日
输出：
  - 3 条差异化选题方向（含竞品分析视角）
  - 每条选题的「第一帧画面」建议（钩子设计）
  - 预估完播率评分（基于三农爆款规律）
```

**任务 B：口播稿升级（强化抖音语言风格）**

当前口播稿偏「文案风」，需要升级为真正的「口语风」：
```
改造点：
  - 开场 3 秒必须有钩子（"你知道为什么×××吗？"）
  - 每 15 秒一个情绪波峰（好奇/惊喜/紧迫）
  - 结尾必须有明确行动指令（"点购物车""马上下单"）
  - 禁止书面语，全部口语化
```

**任务 C：直播话术模块化（可循环使用）**

```
当前：生成一整篇直播话术
升级：模块化设计，商家可自由拼接
  ├─ 开场模块（5 分钟）× 3 套
  ├─ 产品介绍模块（10 分钟/款）× N 款
  ├─ 互动提问模块（3 分钟）× 5 套
  ├─ 紧迫感模块（2 分钟）× 3 套（限时/限量/闪购）
  └─ 催单成交模块（3 分钟）× 3 套
```

#### Day14-Day16：节气/热点内容日历

**任务 D：三农节气营销日历**

```
三农内容的流量密码 = 节气 + 时令 + 农事
内置全年营销节点：
  24 节气 × 对应产品推荐 × 对应口播话术风格
  + 抖音三农热点事件模板（丰收季/助农直播等）

商家每天打开 → 系统自动提示「今天适合推什么」
```

#### Day17-Day20：「今日内容包」前端产品化

```
产品形态：
  - 首屏直接显示「今天是 X 月 X 日 · 适合推：XXX」
  - 一键生成今日内容包（3 分钟）
  - 生成后：每个模块单独复制按钮
  - 历史记录：最近 30 天内容包可回查
```

### Phase 3 · 内测验证（Day21-Day30）

#### Day21-Day23：内测准备

**内测商家画像（精准筛选）：**
```
✅ 目标：月销 5-50 万的抖音三农商家
✅ 有自己在播（不是纯达人合作）
✅ 无专职运营（老板自己或家人在做）
✅ 品类：水果/蔬菜/粮油为主（非农资）
❌ 排除：有专业MCN运营的商家
```

**内测物料：**
- 《免费内测协议》（明确：工具不承诺销量）
- 《3 天快速上手 SOP》（截图版，适合中老年）
- 邀约话术（微信群发版 + 1v1 版）

#### Day24-Day27：陪跑内测

```
内测节奏（每家商家）：
  Day1：注册 + 生成第一个今日内容包 + 1v1 指导使用
  Day3：收集反馈，重点问：「生成的口播稿能直接用吗？」
  Day7：看数据：用了 AI 口播稿的视频 vs 自己写的，完播率对比
```

**核心验证指标：**
```
① 口播稿可直接使用率 ≥ 70%（不需要大改）
② 日活使用率 ≥ 50%（连续 7 天每天打开）
③ NPS ≥ 40（愿意推荐给同行）
```

#### Day28-Day30：付费转化

```
定价策略：
  免费版：今日内容包 3 次/月（体验用）
  基础版：498 元/月（内容包不限次 + 口播稿）
  专业版：1280 元/月（全功能 + 节气日历 + 历史回查）

转化话术核心：
  「你现在每天写稿要 2 小时，一个月 60 小时
    用我们，3 分钟搞定，一个月节省 58 小时
    498 元，相当于你 1 小时的时间成本」

目标：10 家内测 → 转化 ≥ 5 家付费 → 月收 ≥ 2490 元
```

## 五、核心竞争壁垒

### 为什么别人抄不走

```
1. 三农专属知识库（不断迭代沉淀，越用越准）
   通用 AI 生成的内容：「精选优质农产品，新鲜直达」（废话）
   我们生成的内容：「赣南脐橙上市了！霜冻后甜度飙升，糖度 12 度以上，
                    比超市便宜 30%，今天直播只要 29.9！」

2. 节气 × 品类 × 话术三维匹配（三农特有规律）
   通用工具不懂「寒露后柿子最甜」「谷雨前茶最嫩」

3. 用户数据飞轮（哪些话术转化好，自动优化）
   每次商家使用 → 数据沉淀 → 下次生成更准 → 竞品永远追不上
```
