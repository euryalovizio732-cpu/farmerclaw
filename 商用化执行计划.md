## 🔥 P0 — 第一周（7天）：消灭数据空心，4个核心功能达到商用

### Phase 1A：样本库批量填充（Day 1-3）

**策略：公开数据 + LLM结构化提取 + 模板扩展法**

> 核心思路：不爬虫，用3个合规渠道批量获取真实数据

#### 渠道①：公开文章批量提取（LLM辅助）
已有来源还可以深挖：
- 搜狐三农专栏 → 搜索"直播话术""口播稿""农产品带货"
- 知乎专栏 → 搜索"抖音三农""农产品直播""直播带货脚本"
- 微信公众号 → 搜索"农产品直播话术""三农电商"
- 百家号/头条号 → 大量三农MCN公开分享话术

**执行方式**：
1. 我联网搜索并阅读公开文章
2. 用LLM从文章中结构化提取（钩子类型/品类/完整话术）
3. 按格式填入sample_library.py
4. 每条标注来源URL

**目标**：+50条口播稿、+30条选题标题、+20条直播话术、+20条回复

#### 渠道②：行业报告/白皮书数据提取
- 《2025丰收节抖音电商农产品消费白皮书》→ 品类趋势/热搜词
- 《农产品直播电商消费舆情分析报告》→ 痛点结构化数据
- 各省农业厅公开报告 → 品类产量/产地数据
- 巨量算数公开趋势页面 → 热搜词/话题热度

**目标**：+100条痛点数据、+50条热搜词、+20条品类趋势

#### 渠道③：模板扩展法（基于真实样本的变体生成）
- 取已有12条真实口播稿作为"结构母本"
- 用LLM按相同结构生成不同品类变体
- 关键：结构来自真实样本，只替换品类/产地/价格等变量
- 每条标注"结构来源：[真实样本编号]，品类扩展"

**目标**：每条真实样本 → 扩展5个品类变体 = +60条
